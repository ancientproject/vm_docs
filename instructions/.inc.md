# .inc

{% hint style="info" %}
**Note:** float mode support
{% endhint %}

```text
.inc &(0x2)
```

#### Example

```scheme
; Load float value
.orb $(0x1)
.val @float_t(12.3)
.pull &(0x4)

; increment 12.3 -> 13.3
.inc &(0x4)
```

```scheme
; load int
.ldx &(0x1) <| $(0x22)
; increment value 0x22 -> 0x23
.inc &(0x1)
```
